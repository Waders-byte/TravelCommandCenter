# Travel Command Center

## Overview

Travel Command Center is a full-stack travel planning application that helps users organize international travel from a single dashboard.

## Features

- Flight Tracking
- Hotel Comparison
- Budget Management
- Packing Lists
- Itinerary Planning
- Currency Conversion
- Weather
- Interactive Maps

## Built With

- Python
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap
